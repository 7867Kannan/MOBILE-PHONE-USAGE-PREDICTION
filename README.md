This project analyzes daily mobile phone usage patterns using Python and data analytics techniques to determine whether a user is a Normal User, At-Risk User, or Mobile Addict. 
A dataset of 3000+ records is used to train a machine learning prediction model that evaluates time spent across major activity categories such as Social Media, Study, Entertainment, and Gaming.
A web-based application built with Flask allows users to manually enter their daily usage data and receive instant predictions along with insights into their most-used activity. 
To ensure realistic analysis, the system includes input validation enforcing a maximum total usage of 24 hours per day. 
This project demonstrates the complete workflow of data collection, preprocessing, model training, API deployment, and frontend integration, providing a practical solution for behavioral monitoring and digital wellness awareness.
